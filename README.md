# Hanziverse

Hanziverse is a static 3D Chinese language-learning website concept. It includes a Three.js learning universe, HSK 1-6 sidebar navigation, organized vocabulary libraries, video lesson placeholders, and a watch library that links to external drama/media discovery pages by HSK difficulty.

## Run locally

Use any static file server from the repository root. For example:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

> The 3D scene imports Three.js from a CDN, so an internet connection is required for the interactive universe to render.
