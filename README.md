# Hanziverse

Hanziverse is a static marketing website for a Chinese language learning app concept. It presents a Mandarin lesson path, pronunciation practice, hanzi learning, and a graded reading library for learners who want app-style study on the web.

## Run locally

Use any static file server from the repository root. For example:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
